# 🛒 Sistema de Loja em C++

Projeto desenvolvido para a disciplina de **Algoritmos 1**.

O programa simula um sistema simples de loja via terminal, permitindo interação entre **cliente e funcionário**.

## Funcionalidades

👤 Cliente
- visualizar produtos
- adicionar itens ao carrinho
- visualizar carrinho
- finalizar compra
- gerar recibo em arquivo `.txt`

👨‍💼 Funcionário
- adicionar produtos ao estoque
- visualizar estoque
- atualizar preços
- remover produtos

## Tecnologias utilizadas

- C++
- Programação estruturada
- Manipulação de arquivos (`fstream`)
